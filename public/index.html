News api

This api will give you access news and their contents

Topic Users Articles

Get topics

api/topic

Gets a body containing all topics.
e.g. information

description slug img_url
Cooking is love cooking https://shorturl.at/YyqjF

Get users

api/users

Gets a body containing all users.
e.g. information

username name avatar_url
CoolToni2000 Toni https://shorturl.at/8ZQhN

Get articles

api/articles

Gets a body containing all articles.
e.g. information:

title topic autor created_at votes article_img_url article_id
A cool title about cooking cooking Toni 2020-11-22T11:13:00.000Z 12 https://shorturl.at/C2Pio 1

By default the articles are shown in descending order based on their creation date. This behaviour can be changed with the following queries.

  • ?sort_by=[created_at, votes]
  • ?order=[DESC, ASC]

If only articles pertaining to a specific topic are required the following query can be used.

  • ?topic=[topic]

Get article by it's id

api/articles/:article_id

Gets a single article by the given id.
An individualy requested article will, in adition, include the amount of comments made in response to it.
e.g. endpoint

  • /api/articles/3

api/articles/:article_id/comments

Gets the body of all the comments writen to a specific article.
e.g. information:

comment_id body article_id author votes created_at
17 This recipe is alright. 3 CoolToni2000 12 2020-10-03T13:18:00.000Z

By default the comments are shown in descending order.
e.g. endpoint

  • /api/articles/3/comments

A new comment can be added to an article by sending the a username and a message.
It should be formated in the following way.
e.g. { author: 'CoolToni2000', body: 'First comment ever!' }

Delete a comment

/api/comments/:comment

Deletes the indicated article by it's id .
e.g. endpoint

  • /api/comments/11
